# 2D Elevator Simulation (Unity)

## Overview
This project is a 2D Elevator Simulation built using Unity. It demonstrates multiple elevator control systems with smart allocation logic.

## Features
- 3 Elevators system
- 4 Floors (G, 1, 2, 3)
- Floor call buttons
- Nearest elevator selection
- Direction-based smart allocation (Up/Down/Idle)
- Individual request queue for each elevator
- Smooth movement between floors
- UI button enable/disable system

## Logic Implemented
- Elevators respond based on direction and proximity
- Avoids assigning multiple elevators to same request
- Idle elevators are used when needed
- Queue-based request handling

## Technologies Used
- Unity Engine 6
- C#
- Unity UI

## How to Run
1. Open project in Unity Hub
2. Open the main scene
3. Click Play
4. Use buttons to call elevators
